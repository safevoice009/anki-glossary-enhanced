## 🚀 Quick Start & Installation

### Option 1: One-Click Zip Installation (Recommended)
1. Download **[EnterMedSchool_Glossary_Enhanced_Edition.zip](https://github.com/safevoice009/anki-glossary-enhanced/releases/latest)**.
2. Open Anki → **Tools > Add-ons > View Files**.
3. Unzip the downloaded folder into your Anki add-ons directory.
4. Restart Anki!

### Option 2: AnkiWeb Add-on Code
1. Open Anki → **Tools > Add-ons > Get Add-ons...**
2. Enter the AnkiWeb Add-on Code.
3. Click **OK** and restart Anki.

## 🛠️ How To Use During Review

1. **Review Cards**: Open any deck and start reviewing.
2. **Double-Click Any Word**: Double-click any medical term, drug, symptom, or syndrome on your card screen.
3. **Instant Medical Popup**: The EnterMedSchool Glossary window will slide open with the full definition, mechanism, clinical presentation, and high-yield exam traps!
4. **Move & Position**: Drag the window header to position it anywhere on your desktop or secondary monitor.
